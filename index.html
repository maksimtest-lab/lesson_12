Урок 12

Задание 1

  • Установите Git на свой компьютер, если он ещё не установлен.
  • Создайте локальный репозиторий Git для проекта, с которым вы будете работать.
  • Создайте новый файл (например, index.html) и добавьте в него какой-то простой HTML-код.
  • Инициализируйте Git в этой папке и создайте первый коммит с вашим файлом.

Задание 2

  • Создайте новую ветку с именем "feature" (или любым другим именем, которое вам нравится).
  • Внесите изменения в файл index.html (например, добавьте заголовок или измените содержимое).
  • Закоммитьте ваши изменения в этой ветке.
  • Переключитесь обратно на основную ветку (обычно это "main" или "master") и убедитесь, что ваш файл не содержит изменений.
  • Слейте изменения из ветки "feature" в основную ветку с помощью git merge.

Задание 3

  • Зарегистрируйтесь на GitHub (если вы еще не зарегистрированы).
  • Создайте новый удалённый репозиторий на GitHub.
  • Свяжите ваш локальный репозиторий с удалённым репозиторием на GitHub.
  • Отправьте вашу локальную ветку "feature" на удалённый репозиторий с помощью git push.
  • Убедитесь, что ваши изменения появились на GitHub.
